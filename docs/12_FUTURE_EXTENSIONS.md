# FUTURE_EXTENSIONS: 살만해

## 등기부등본 AI 분석

사용자가 등기부등본 PDF 또는 이미지를 업로드하면 AI가 읽어서 계약 위험 요소를 요약합니다.

```
소유자: 홍길동
근저당권: 있음 / 채권최고액: 2억 4천만 원
압류/가압류: 없음
신탁등기: 없음
위험 포인트: 근저당 말소 확인 필요
```

- Claude Vision API로 이미지/PDF에서 텍스트 추출
- 갑구(소유권), 을구(근저당·압류·신탁) 자동 파싱
- 위험 요소 요약 + 다음 행동 안내
- F-3 법률 RAG 챗봇과 연계: "이 근저당 어떻게 해야 해?" 후속 질문 가능

## 실제 매물 수급

MVP는 더미 매물을 사용하지만, 이후 실제 매물로 전환합니다.

- 네이버부동산 크롤링 (법적 검토 필요)
- 공인중개사 API 제휴
- 직접 매물 등록 기능 (중개사 계정)

## 정교한 HUG/HF/SGI 판정

간이 계산에서 벗어나 등기부등본 데이터 기반 정밀 판정으로 확장합니다.

- 선순위채권 자동 파싱 (등기부등본 연동)
- HUG·HF·SGI 기준별 상세 비교
- 공시가격 자동 조회 연동

## 지역 커뮤니티

공공데이터로 알 수 없는 실거주 정보를 공유하는 기능입니다.

- 지역/건물 후기 작성
- 치안 만족도, 소음, 벌레, 관리 상태 후기
- 게시글 신고/수정/삭제
- 건물 단위 리뷰 모아보기

## 개인화 추천

찜·조회 이력을 활용해 추천을 고도화합니다.

- 찜한 매물 기반 유사 매물 추천
- 조회 이력 기반 선호 조건 추론
- 임베딩 기반 유사 매물 검색
