# PRD: 살만해

## 목표
청년 1인 가구가 AI 에이전트와 지도로 부동산 매물을 탐색하고 "이 계약이 안전한가"를 판단할 수 있는 서비스

## 핵심 기능 (MVP)

1. **지도 기반 매물 탐색** — 마커·클러스터링·필터(거래유형·매물유형·가격), 실거래가 레이어, 안전 레이어 (비로그인 포함)
2. **AI 에이전트 — 매물 추천** — 자연어 입력 → 의도 분류 → 지역·예산·유형 파싱 → 최대 5개 매물 카드 반환, 지도 마커 연동 (로그인 필요)
3. **AI 에이전트 — 법률 RAG** — 임대차보호법·전세사기특별법 질의응답, 법령 조항 원문 + AI 해설 (로그인 필요)
4. **시세·안전 분석** — 실거래가 추이·동일 평형 비교, 반경 안전시설 점수화 (기본 조회는 비로그인 가능)
5. **인증** — 이메일/비밀번호 회원가입·로그인, JWT 기반 세션
6. **찜하기** — 매물 저장, 찜 목록 조회, 지도 별도 마커 표시

## MVP 제외 사항

- 실제 매물 중개 (더미 매물로 대체, 중개사 매물 등록 없음)
- 커뮤니티 (게시글, 신고, 권한)
- 뉴스 RAG (1.5차 — 딥서치 API 크롤링·재인덱싱 필요)
- HUG/HF/SGI 보증 정밀 판정 (1.5차 — 공시가격 API 연동 필요)
- 등기부등본 AI 분석 (확장 — Claude Vision API)
- 소셜 로그인 (1.5차 — 네이버 OAuth2.0)
- 대화 세션 및 요약 메모리 (1.5차 — LangGraph 상태 관리)
- 앱 / 모바일 네이티브 버전

## 사용자 권한

| 상태 | 접근 가능 기능 |
|------|--------------|
| 비로그인 | 지도 탐색, 필터, 실거래가 레이어, 안전 레이어, 기본 시세·안전 조회 |
| 로그인 | 위 전체 + AI 에이전트 (추천·법률·분석), 찜하기 |
