# 04. UI_GUIDE

## 핵심 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  상단 헤더: 로고 + 검색창 + 로그인/프로필            │
├──────────────────┬──────────────────────────────────┤
│  왼쪽 패널       │                                  │
│  (AI 챗봇 +      │        네이버지도               │
│   결과 카드)     │  (매물 마커, 레이어 토글)        │
│  340px 고정      │                                  │
└──────────────────┴──────────────────────────────────┘
```

- 왼쪽 패널은 항상 표시, 지도가 나머지 영역을 채운다.
- 모바일 반응형은 MVP 범위 외 (데스크톱 우선).

## 색상 팔레트

| 역할 | 색상 | 사용처 |
|------|------|--------|
| Primary | `#2563EB` (Blue 600) | CTA 버튼, 활성 마커, 링크 |
| Danger | `#EF4444` (Red 500) | 위험 매물, 에러 상태 |
| Safe | `#22C55E` (Green 500) | 안전 점수 높음, 성공 |
| Warning | `#F59E0B` (Amber 500) | 안전 점수 중간, 경고 |
| Neutral | `#6B7280` (Gray 500) | 비활성 텍스트, 보조 정보 |
| Surface | `#F9FAFB` (Gray 50) | 카드 배경, 패널 배경 |

## 컴포넌트 패턴

- 매물 카드: 이미지 없음 (더미 매물), 건물명·주소·거래유형·가격·평수·층수·안전점수 표시
- AI 응답 버블: 사용자(오른쪽 파란 버블) / AI(왼쪽 흰 버블 + 아이콘)
- 의도 배지: 챗봇 상단에 현재 모드 표시 (매물 추천 / 법률 상담 / 시세 분석 / 안전 분석)
- 법령 참조 카드: 왼쪽 패널 하단, 조항명 + 원문 요약 + 접기/펼치기

## 안티패턴 (AI 슬롭 — 절대 사용 금지)

- Glass morphism (backdrop-filter blur 남용)
- 보라색·핑크 그라데이션 텍스트 (`bg-clip-text text-transparent gradient`)
- 네온 글로우 (`drop-shadow` + 형광색)
- 카드 모든 곳에 `rounded-2xl` + `shadow-2xl` 남발
- 헤더에 별 의미 없는 hero gradient 배너
- 이모지를 버튼 레이블에 삽입 (아이콘은 lucide-vue-next 사용)

## 아이콘

lucide-vue-next만 사용한다. 인라인 SVG나 다른 아이콘 라이브러리 혼용 금지.

## 지도 마커 색상

| 상태 | 색상 |
|------|------|
| 기본 매물 | Blue 600 |
| 찜한 매물 | Amber 400 |
| AI 추천 매물 | Green 500 |
| 현재 선택 | Red 500 (강조) |

## Tailwind 클래스 규칙

- 반응형 prefix는 MVP에서 사용하지 않는다 (`sm:`, `md:`, `lg:` 등).
- 커스텀 색상은 tailwind.config.js에 등록하지 않고 표준 팔레트만 사용한다.
- 컴포넌트 스타일은 scoped `<style>` 대신 Tailwind 클래스로 처리한다.
