# 01. PRD

## 서비스 정의

**살만해**는 AI 에이전트 기반 부동산 탐색 서비스입니다.

사용자가 자연어로 조건을 입력하면, AI 에이전트가 매물·시세·안전정보·계약 법률 정보를 연결해 추천과 설명을 제공합니다. 전월세·매매를 모두 다루며, 중개 서비스가 아닌 **계약 전 안전성 검증 중심** 서비스입니다.

## 해결하려는 문제

- 청년 1인 가구는 전세사기·깡통전세·허위매물 위험에 취약합니다.
- 등기부등본·선순위채권·HUG 보증 기준 같은 계약 안전 정보를 스스로 해석하기 어렵습니다.
- 실거래가·안전시설·법률 정보가 여러 곳에 흩어져 있어 한 번에 확인하기 어렵습니다.
- 기존 부동산 서비스는 매물 탐색에 집중하며 "이 계약이 안전한가"에 대한 답을 주지 않습니다.

## 핵심 방향

살만해의 메인은 **AI 에이전트와 지도의 연동**입니다.

```
지도에서 매물 탐색 (비로그인)
→ AI 에이전트에 자연어로 질문 (로그인)
    ├─ 매물 추천 → 지도 마커 연동
    ├─ 시세·안전 분석
    └─ 법률 상담 (임대차보호법, 전세사기특별법)
→ HUG 보증 간이 자가진단
→ "이 집 계약해도 되나?" 판단
```

## 서비스 포지셔닝

| 서비스 | 강점 | 살만해와 차이 |
|--------|------|-------------|
| 네이버부동산·직방 | 매물 탐색·중개 | 살만해는 중개 없음, 안전 검증 중심 |
| 호갱노노 | 실거래가·시세 | 살만해는 AI 에이전트 + 법률 RAG 추가 |

## 핵심 기능 (MVP)

1. **지도 기반 매물 탐색** — 마커·클러스터링·필터(거래유형·매물유형·가격), 실거래가 레이어, 안전 레이어 (비로그인 포함)
2. **AI 에이전트 — 매물 추천** — 자연어 입력 → 의도 분류 → 지역·예산·유형 파싱 → 최대 5개 매물 카드 반환, 지도 마커 연동 (로그인 필요)
3. **AI 에이전트 — 법률 RAG** — 임대차보호법·전세사기특별법 질의응답, 법령 조항 원문 + AI 해설 (로그인 필요)
4. **시세·안전 분석** — 실거래가 추이·동일 평형 비교, 반경 안전시설 점수화 (기본 조회는 비로그인 가능)
5. **인증** — 이메일/비밀번호 회원가입·로그인, JWT 기반 세션
6. **찜하기** — 매물 저장, 찜 목록 조회, 지도 별도 마커 표시

## 사용자 권한

| 상태 | 접근 가능 기능 |
|------|--------------|
| 비로그인 | 지도 탐색·필터, 실거래가 레이어, 안전 레이어, 기본 시세·안전 조회 |
| 로그인 | 위 전체 + AI 에이전트 (추천·법률·분석), 찜하기 |

## MVP 제외 사항

| 기능 | 제외 이유 |
|------|----------|
| 실제 매물 중개 | 더미 매물로 대체. 중개사 등록·문의·거래 관리는 후순위. |
| 커뮤니티 | 게시글·신고·권한 처리 범위가 큼. |
| 뉴스 RAG | 크롤링 주기 관리·재인덱싱 필요. 1.5차. |
| HUG 정밀 판정 | 공시가격 API 연동 + 사용자 입력 복잡도. 1.5차. |
| 등기부등본 AI 분석 | Claude Vision API 연동 필요. 확장 단계. |
| 소셜 로그인 | OAuth 연동 복잡도. 1.5차. |
| 대화 세션·메모리 | LangGraph 상태 관리 복잡도. 1.5차. |
